# DevArc

DevArc is an AI-powered platform designed to help developers master coding interviews and accelerate their career growth.

## Vision

Build the developer career arc using AI.

## Core Modules

- Coding practice platform
- AI hints and explanations
- AI code review
- Interview preparation
- Career roadmap generation

## Architecture

DevArc is designed as a modular platform:

frontend
backend
ai services
infrastructure

## Tech Stack

Frontend
- Next.js
- Tailwind

Backend
- Node.js
- Express

Database
- PostgreSQL

AI
- OpenAI / Azure OpenAI
